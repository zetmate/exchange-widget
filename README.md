# Exchange Widget
A small application for exchanging currencies.

## Getting started

### 1. Install the packages:
```
yarn
```

### 2. Run the app:
```
yarn start
```

## Description
At the start you have an initial balance. To exchange the currencies go to exchange screen by clicking the "Exchange" button.
All of your operations will appear in the history. The latest operations are shown first.
Note that hanges are not stored in the browser storage and will be lost after a page refresh.
